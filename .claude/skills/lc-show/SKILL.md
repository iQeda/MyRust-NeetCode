# LeetCode Show

Show problem details and generate Rust template.

## Usage

`/lc-show <problem_id>` - Show problem and generate Rust code template

## Instructions

1. Run `leetcode show <problem_id> 2>/dev/null` to display the problem description
2. Run `leetcode show <problem_id> -g -l rust 2>/dev/null` to generate the Rust template
3. Show both results to the user
