# 計算量の基礎

## 計算量とは

「入力サイズが増えたとき、処理時間がどれくらい増えるか」を表す指標。

実際の秒数ではなく、**増加の仕方（オーダー）** を表す。

## Big O 記法

`O(n)` のように書く。`n` は入力サイズ。

```
O(1)      定数時間
O(log n)  対数時間
O(n)      線形時間
O(n log n)
O(n²)     二乗時間
O(2ⁿ)     指数時間
```

## 増加の速さを比較

n = 1,000,000（100万）の場合:

| 計算量 | 処理回数 | 実行時間の目安 |
|--------|----------|----------------|
| O(1) | 1 | 一瞬 |
| O(log n) | 20 | 一瞬 |
| O(n) | 1,000,000 | 0.01秒 |
| O(n log n) | 20,000,000 | 0.2秒 |
| O(n²) | 1,000,000,000,000 | **数時間** |
| O(2ⁿ) | 宇宙の原子数を超える | **不可能** |
