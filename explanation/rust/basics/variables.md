# 変数

## 不変と可変

```rust
let x = 5;          // 不変（immutable）- 変更不可
let mut y = 10;     // 可変（mutable）- 変更可能

y = 20;             // OK
// x = 10;          // エラー！
```

## なぜデフォルトが不変？

- バグを防ぐ
- コードが読みやすくなる
- 変更が必要な変数が明確になる

## シャドーイング

```rust
let x = 5;
let x = x + 1;      // 新しい x を作る（上書きではない）
let x = "hello";    // 型も変えられる
```
