# 217. Contains Duplicate

## 問題

整数配列 `nums` が与えられたとき、同じ値が2回以上出現するなら `true`、全て異なるなら `false` を返す。

```
Input: nums = [1,2,3,1]
Output: true  // 1 が2回出現

Input: nums = [1,2,3,4]
Output: false // 全て異なる
```

## 解法: HashSet

### アイデア

HashSet に要素を追加していき、既に存在する要素があれば重複。

### コード解説

```rust
let mut seen = HashSet::new();

for num in nums {
    if !seen.insert(num) {  // insert は既存なら false を返す
        return true;        // 重複発見
    }
}
false  // 重複なし
```

### HashSet::insert の挙動

```rust
seen.insert(1);  // true  (新規追加)
seen.insert(2);  // true  (新規追加)
seen.insert(1);  // false (既に存在)
```

`insert` は:
- 新規追加: `true` を返す
- 既存: `false` を返し、何もしない

この性質を利用して、1行で「追加 + 重複チェック」を行う。

### 計算量

- 時間: O(n) - 配列を1回走査、HashSet の insert/contains は O(1)
- 空間: O(n) - HashSet に最大 n 個の要素

### 別解

| 解法 | 時間 | 空間 |
|------|------|------|
| 全探索 (2重ループ) | O(n²) | O(1) |
| ソート後に隣接比較 | O(n log n) | O(1)* |
| **HashSet** | **O(n)** | O(n) |

*ソート解法は in-place なら O(1)、そうでなければ O(n)
